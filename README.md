# Pets

A small collection of shareable custom Codex pets.

## Included pets

### Star Trek / Data-inspired android officer

[`star-trek/data`](star-trek/data) contains a Codex v2 animated pet: a composed, golden-skinned android officer inspired by the Lt. Commander Data character archetype.

To install it locally:

```bash
mkdir -p ~/.codex/pets/data
cp star-trek/data/pet.json star-trek/data/spritesheet.webp ~/.codex/pets/data/
```

Restart or refresh Codex after copying the files.
